# Swathi Dhanasekaran

**Senior Data Scientist - Bioinformatics | Computational Biology | Reproducible Omics | AI/ML for Translational Research**

I build reproducible bioinformatics workflows that connect raw omics data to biological interpretation, target discovery, and decision-ready scientific evidence. My work spans NGS pipeline development, multi-omics integration, regulatory genomics, biomarker discovery, and applied machine learning for biomedical research.

Currently, I am building **Bioinformatics Field Guide**, a public knowledge platform for practical bioinformatics: clear methods, reusable notebooks, reproducible environments, paper reviews, and decision frameworks for learners, researchers, and industry teams.

## Featured Projects

### [FibroTarget-Liver](https://github.com/Caffeinated-Code/FibroTarget-Liver)

Reproducible single-cell workflow for human liver fibrosis, MASH, and cirrhosis target discovery. Includes a Seurat-based analysis, public-dataset validation, target scoring, executive reporting, Shiny dashboard, and local/AWS Nextflow demo.

**Why it matters:** This is the flagship portfolio project because it combines biological interpretation, reproducible infrastructure, multi-dataset validation, and translational target prioritization.

### [Mishra_IL10_ATACseq_DownstreaAnalysis](https://github.com/Caffeinated-Code/Mishra_IL10_ATACseq_DownstreaAnalysis)

Downstream ATAC-seq analysis of Mishra et al. GSE261244, focused on epigenomic interpretation and reproducible reporting.

**Why it matters:** Shows practical chromatin-accessibility analysis, public dataset reuse, and communication of epigenomics results.

### [Precision_Oncology](https://github.com/Caffeinated-Code/Precision_Oncology)

Analysis project focused on enriched mutations in drug-responsive patients.

**Why it matters:** Connects genomic data analysis to clinical and therapeutic interpretation.

### [MIT_Applied_DataScience](https://github.com/Caffeinated-Code/MIT_Applied_DataScience)

Applied AI/ML coursework and biomedical imaging capstone from MIT Professional Education. The capstone used OpenCV preprocessing and CNN architectures in TensorFlow/Keras for microscopy/RBC image analysis, reaching 97% accuracy and 99% sensitivity for malaria detection in the project setting.

**Why it matters:** Demonstrates applied machine learning, computer vision, and the ability to translate AI methods into biomedical analysis workflows.

### [advanced-bio-sql-engineering](https://github.com/Caffeinated-Code/advanced-bio-sql-engineering)

Advanced SQL case study using public database APIs for insulin pathway data mining and structured biological knowledge exploration.

**Why it matters:** Shows database thinking, public data engineering, and biological query design beyond notebook-only analysis.

### [TheAyurPhyta](https://github.com/Caffeinated-Code/TheAyurPhyta)

Early large-scale data retrieval and database-building project using public phytochemical resources, scripting, parsing, and SQL.

**Why it matters:** Demonstrates long-standing interest in biological databases, data wrangling, and computable biological knowledge.
